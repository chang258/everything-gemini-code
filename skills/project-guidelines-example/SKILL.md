# Project Guidelines Skill (Template)

此技能作為建立專案專屬規範的範本，涵蓋了從前端到部署的完整技術棧定義。

## 專案要素
- **架構概覽**: 定義服務間的通訊路徑（如 Frontend <-> Backend <-> AI API）。
- **目錄規範**: 標準化各層級（App, Components, Services, Tests）的存放位置。
- **代碼慣例**: 例如「嚴禁在代碼中使用 Emoji」、「強制使用不可變物件」。

## 質量門檻
- **TDD 優先**: 代碼實作前必須先有測試。
- **覆蓋率**: 強制要求全系統覆蓋率達 80% 以上。
- **檔案體積**: 單個檔案建議維持在 200-400 行之間。
